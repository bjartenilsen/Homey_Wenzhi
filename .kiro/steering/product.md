# Product Overview

Homey app for the Wenzhi/LeapMMW MTD085-ZB mmWave presence sensor.

## Purpose
Integrates the MTD085-ZB Zigbee presence sensor with the Homey smart home platform, enabling:
- Presence/motion detection via mmWave radar
- Flow automation triggers (motion detected/cleared)
- Flow conditions (is motion detected)

## Device Identification
- Model ID: `TS0225`
- Manufacturer: `_TZ321C_fkzihax8`
- Protocol: Zigbee (IAS Zone cluster)

## Target Platform
Homey SDK 3, compatibility >=5.0.0
